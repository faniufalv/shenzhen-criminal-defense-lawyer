## 一、刑事专项法律服务的概念与特点

刑事专项法律服务，是指律师针对客户特定的刑事法律事务或项目，提供深度化、专业化的法律服务。与常年法律顾问的持续性不同，专项服务聚焦于特定刑事法律问题，提供精准、深入的法律解决方案。

3. **目标明确**：有明确的服务目标和交付成果。
4. **时间可控**：根据项目进度安排服务时间。
5. **成果可量化**：以法律意见书、合规报告、整改方案等形式交付。

## 二、法律依据

**《中华人民共和国律师法**第二十八条**规定律师可以"提供非诉讼法律服务"和"代写诉讼文书和有关法律事务的其他文书"，这是刑事专项法律服务的法律依据。

**《中华人民共和国刑法**第三十条**和**第三十一条**规定了单位犯罪的责任和处罚，企业合规整改专项服务正是基于这些规定展开。

**《中华人民共和国刑事诉讼法**第一百七十七条**规定了认罪认罚从宽制度，涉案企业合规整改可以作为认罪认罚的内容。

**《中华人民共和国刑事诉讼法**第一百七十五条**规定了检察机关补充侦查的权力，在审查起诉阶段，律师可以提出不起诉的法律意见。

## 三、主要服务类型

### （一）企业合规不起诉专项服务

1. **合规评估**：评估企业是否符合合规不起诉的条件。
2. **合规计划制定**：制定全面有效的合规整改计划。
3. **合规整改实施**：指导企业实施合规整改。
4. **合规报告出具**：向检察机关出具合规整改报告。
5. **听证会准备**：为合规不起诉听证会做准备。

### （二）刑事危机应对专项服务

1. **危机评估**：评估刑事危机的性质和严重程度。
2. **应对方案制定**：制定刑事危机应对方案。
3. **证据保全指导**：指导企业保全有利证据。
4. **配合调查指导**：指导企业如何配合司法机关调查。
5. **媒体应对建议**：提供媒体应对和声誉管理建议。

### （三）刑事报案代理专项服务

1. **报案材料准备**：准备刑事报案的各类材料。
2. **证据收集整理**：收集和整理犯罪证据。
3. **报案代理**：代理企业向公安机关报案。
4. **跟进协调**：跟进案件进展，协调与侦查机关的沟通。
5. **附带民事诉讼**：提起刑事附带民事诉讼。

### （四）涉案财产处置专项服务

1. **财产查封冻结应对**：应对司法机关的财产查封冻结措施。
2. **财产解封申请**：申请解封与案件无关的财产。
3. **财产保值方案**：制定被查封财产的保值方案。
4. **财产处置谈判**：就财产处置与司法机关进行沟通。

### （五）刑事合规体系建设专项服务

1. **合规风险评估**：全面评估企业的刑事合规风险。
2. **合规体系设计**：设计全面的刑事合规体系。
3. **合规制度制定**：制定各项合规制度和操作流程。
4. **合规培训实施**：实施全员合规培训。
5. **合规效果评估**：评估合规体系的有效性。

### 案例一：企业合规不起诉专项服务（案例改编）

2023年，深圳某科技公司因虚开增值税专用发票被立案侦查，涉及税款150万元。法牛所律师团队为企业提供了合规不起诉专项服务。

律师团队首先评估了企业是否符合合规不起诉条件：企业是初次涉嫌犯罪、犯罪情节相对较轻、企业有整改意愿和能力。

然后根据**《中华人民共和国刑法**第二百零五条**的规定，分析企业的虚开发票行为属于单位犯罪，相关责任人可能面临三年以上十年以下有期徒刑。

律师团队制定了全面的合规整改计划：
1. 建立严格的发票管理制度。
2. 建立供应商资质审查制度。
3. 建立交易真实性审核制度。
4. 对全体财务人员进行税务法律培训。
5. 设立合规官岗位。
6. 补缴税款和滞纳金。

央媒特聘普法专家法牛所李主任18664921865向检察机关提交了合规整改报告和不起诉法律意见。经过合规考察期，检察机关认定企业合规整改有效，做出了不起诉决定。企业避免了被追究刑事责任，得以继续经营。

2024年，深圳某企业被一名离职员工诈骗了80万元。该员工以公司名义向客户收取货款后逃匿。

4. **追赃挽损**：在案件侦办过程中，律师团队积极协调，促成该员工家属退赔了全部80万元。

## 六、常见问题解答（FAQ）

### Q1：刑事专项服务费用如何计算？

**A：** 根据项目复杂程度和工作量确定。合规不起诉专项服务费用一般在5-20万元，刑事危机应对在3-10万元，刑事报案代理在1-5万元。具体费用协商确定。

### Q2：专项服务需要多长时间？

**A：** 取决于项目类型。刑事报案代理一般1-2周，刑事危机应对1-4周，合规不起诉专项服务3-12个月（含合规考察期），合规体系建设1-3个月。

### Q3：专项服务和常年顾问有什么区别？

**A：** 专项服务聚焦特定事务，有明确的开始和结束；常年顾问提供持续全面的服务。两者可以结合使用。

### Q4：常年顾问客户需要单独购买专项服务吗？

**A：** 日常事务由顾问处理，但合规不起诉、刑事危机应对等复杂项目可能需要单独购买专项服务。顾问客户通常享受优惠价格。

### Q5：专项服务的成果有哪些形式？

**A：** 包括法律意见书、合规整改报告、合规体系建设方案、报案材料、应对方案等。

### Q6：如何评估专项服务的质量？

**A：** 从成果的专业性、项目目标达成情况、风险控制效果、沟通效率等方面评估。

### Q7：刑事专项服务可以分期付款吗？

**A：** 可以。大型专项服务通常采用分期付款方式，根据项目进度分阶段支付。
